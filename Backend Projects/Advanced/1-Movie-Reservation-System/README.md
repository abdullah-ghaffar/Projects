# 🎬 Movie Reservation CLI

A pure Node.js command-line tool to book, view, and cancel movie tickets.

---

## Commands

### List Movies

`node movie-cli.js list`

### Book a Ticket

`node movie-cli.js book <movie_id> "<Your Name>"`

### View Your Bookings

`node movie-cli.js view "<Your Name>"`

### Cancel a Ticket

`node movie-cli.js cancel <ticket_id>`

---

## Features

- Zero external libraries.
- Full CRUD for ticket management.
- Data is stored locally in JSON files.
