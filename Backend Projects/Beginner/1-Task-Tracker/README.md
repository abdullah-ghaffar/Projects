# 🚀 Task Tracker CLI

A lightweight, pure Node.js Command Line Interface tool to manage your daily tasks with a local JSON database.

## 🛠 Features
- **Zero External Dependencies**
- **Full CRUD operations** (Create, Read, Update, Delete)
- **Status Tracking** (Todo, In-Progress, Done)

## 💻 Usage
- **Add Task:** `node task-cli.js add "My new task"`
- **List All:** `node task-cli.js list`
- **List Done:** `node task-cli.js list done`
- **Update Task:** `node task-cli.js update <id> "New name"`
- **Delete Task:** `node task-cli.js delete <id>`
